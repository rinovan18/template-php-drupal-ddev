EZContent Publish Ckeditor Layoutmanager Feature

Description:
------------
Module add ckeditor layoutmanager plugin in to full html editor mode,
Ckeditor layoutmanager manager comes with predefined layouts based on
twelve column layout of bootstrap.

Todo: Contribute this module to drupal community.

Usage:
------
After installing and activating the 'EZContent Publish Ckeditor Layoutmanager
Feature' module you should be able to see layoutmanager icon in full html editor
mode.If you wish to change the full html editor config go to
 admin/config/content/formats/manage/full_html
