Ckeditor Layout Manager 

Description:
------------
Module add ckeditor Layout manager plugin, which allow content editor or
site builder to add different layout inside WYSIWYG editors (basic html, full html etc),
there are predefined layout provided by plugin based on twelve column layout of bootstrap.
In order to use this module you should have a bootstrap based front end theme.

Dependency:
------------
This plugin requires bootstrap based theme, as it uses bootstrap classes to render layouts.



Usage:
------
After installing and activating the 'Ckeditor Layout Manager'
module you should be able to see layoutmanager plugin icon (AddLayout) in text
editor settings modes.Visit admin/config/content/formats and add layoutmanager
icon in any of the editor mode i.e Basic Html or Full Html.
