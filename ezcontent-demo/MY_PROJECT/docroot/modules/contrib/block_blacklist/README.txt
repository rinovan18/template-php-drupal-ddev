INTRODUCTION
------------

The Block Blacklist module removes unnecessary blocks from the block list for better
system performance.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/block_blacklist

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/block_blacklist


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://www.drupal.org/node/1897420 for further information.


CONFIGURATION
-------------

    1. Navigate to Administration > Extend and enable the module.
    2. Navigate to Administration > Configuration > System > Block Blacklist Settings and
       list each name or value on a new line in the appropriate section. The user has the
       option to identify blocks to be removed by name or prefix, or provide a regex for
       more complex matching options.
    3. Save.
